##### 以下为Linux下的32 位C 程序，请计算sizeof 的：
（4）`void Func ( char str[100])                           
 {                                          
 …… ;                                                            
 }  ;`  
##### 请计算  
请计算`sizeof( str ) =  `    
### 【标准答案】
#### （4）4
### 【答案解析】
指针即为地址，指针几个字节跟语言无关，而是跟系统的寻址能力有关，  
譬如以前是16为地址，指针即为2个字节，  
现在一般是32位系统，所以是4个字节，  
以后64位，则就为8个字节。
